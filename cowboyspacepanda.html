Cowboy Space Panda

I initially prototyped this game as a project for an introduction to game programming course in college. I have since built on it extensively and released it as a more complete product.

Check it out here in the browser, or on itch.io! Alternatively, you can run the game locally by downloading the source code from my GitHub and opening the index.html file.

The game can be played using the keyboard, and supports up to two players on a single device.

Use (WASD) to move and (Space) to select.

You are a Space Panda, the last of your kind, set on collecting all the rainbow stars from outer space -- But look out for meteors! Space is a dangerous place.
